# 학습 흐름 초안

이 앱은 `신규 학습`과 `복습 학습`을 분리해서 운영한다.

## 핵심 원칙

- 신규 학습은 `개념 -> 암기 100% 통과 -> 해당 lesson 기출 -> 다음 lesson` 순서로 진행한다.
- 복습 학습은 `오늘 due 된 기출 문제`만 복습 큐에서 꺼내서 처리한다.
- 암기 문항은 신규 학습 단계의 통과 게이트로만 사용하고, 복습 큐에는 넣지 않는다.
- 기출문제 풀이 UI는 신규 기출, 오답노트, due 복습, 모의고사 오답 복습에 동일하게 사용한다.
- 하루 학습량은 `신규 학습량 + 기출 복습 학습량`으로 본다.

## 전체 흐름

```mermaid
flowchart TD
    A["앱 시작"] --> B["오늘 학습 상태 계산"]
    B --> C["오늘의 신규 학습 확인"]
    B --> D["오늘의 복습 학습 확인"]

    C --> E["현재 과목 / 현재 lesson 진입"]
    E --> F["개념 학습"]
    F --> G["암기 문제 풀이"]
    G --> H{"암기 100% 통과?"}
    H -- "아니오" --> I["같은 lesson 개념 학습으로 복귀"]
    I --> F
    H -- "예" --> J["해당 lesson 기출 문제 풀이"]
    J --> K["lesson 완료"]
    K --> L{"다음 lesson 존재?"}
    L -- "예" --> E
    L -- "아니오" --> M["다음 과목으로 이동"]

    D --> N{"오늘 due 기출 존재?"}
    N -- "예" --> O["기출 복습 큐 진입"]
    O --> P["기출 문제 풀이"]
    P --> Q["채점"]
    Q --> R{"정답 여부"}
    R -- "정답" --> S["다음 복습일 재설정"]
    R -- "오답" --> T["우선순위 상승 및 가까운 복습일 재설정"]
    S --> U{"남은 due 기출 있음?"}
    T --> U
    U -- "예" --> P
    U -- "아니오" --> V["오늘 복습 완료"]

    M --> W{"전 과목 신규 1회전 완료?"}
    W -- "아니오" --> E
    W -- "예" --> X["모의고사 준비"]

    V --> X
    X --> Y["모의고사 풀이"]
    Y --> Z["채점 및 오답 등록"]
    Z --> AA["모의고사 오답 기출 복습 큐 등록"]
```

## 신규 학습 상세

```mermaid
flowchart TD
    A["lesson 시작"] --> B["개념 학습"]
    B --> C["개념 학습 완료 클릭"]
    C --> D["암기 문제 시작"]
    D --> E["암기 문항 풀이"]
    E --> F["정답 확인"]
    F --> G{"다음 암기 문항 존재?"}
    G -- "예" --> E
    G -- "아니오" --> H{"전체 100% 정답?"}
    H -- "아니오" --> I["암기 세션 채점 초기화"]
    I --> J["같은 lesson 개념 학습으로 복귀"]
    H -- "예" --> K["해당 lesson 기출 문제 시작"]
    K --> L["기출 1문항 풀이"]
    L --> M["채점"]
    M --> N["결과 표시"]
    N --> O{"다음 기출 문항 존재?"}
    O -- "예" --> L
    O -- "아니오" --> P["lesson 완료"]
    P --> Q["다음 lesson 이동"]
```

## 복습 학습 상세

```mermaid
flowchart TD
    A["복습 시작"] --> B["오늘 due 기출 조회"]
    B --> C{"due 기출 있음?"}
    C -- "아니오" --> D["복습 없음"]
    C -- "예" --> E["우선순위 순 정렬"]
    E --> F["기출 공통 풀이 UI 진입"]
    F --> G["문항 풀이"]
    G --> H["채점"]
    H --> I{"정답?"}
    I -- "예" --> J["다음 복습일 뒤로 조정"]
    I -- "아니오" --> K["가까운 날짜로 재예약"]
    K --> L["오답 상태 유지"]
    J --> M{"남은 due 기출 있음?"}
    L --> M
    M -- "예" --> F
    M -- "아니오" --> N["오늘 복습 완료"]
```

## 복습 큐 규칙

- 기출 정답: `1일 -> 7일`
- 기출 오답: `당일 -> 1일 -> 3일 -> 7일`
- 같은 문항이 이미 큐에 있으면 새 항목을 계속 만들지 않고 기존 due와 상태를 갱신한다.

## 화면별 역할

- 홈: 오늘의 신규 학습, 오늘의 복습 학습, due 수량, 권장 신규 물량 표시
- 개념학습: lesson 개념 확인 후 암기로 이동
- 암기학습: 신규 암기만 처리
- 기출문제: 신규 기출, due 복습, 오답노트, 모의고사 오답 복습에 공통 UI 사용
- 오답노트: 누적 오답 기록과 due 기출 복습 확인
- 학습현황: 신규 진도, 오늘 기출 복습량, 과목별 취약도 확인
- 설정: 30일/60일 플랜 선택, 학습 기록 초기화
